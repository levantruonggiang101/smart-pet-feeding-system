# Smart Pet Feeding System Using ESP32

## Overview
An IoT-based Smart Pet Feeding System that automatically dispenses food and manages water supply for pets.

## Features
- Automatic feeding schedule
- Food weight monitoring
- Water level monitoring
- OLED display
- Mobile app control
- WiFi communication

## Hardware
- ESP32 DevKit V1
- Load Cell + HX711
- Water Level Sensor
- RTC DS1307
- OLED Display
- Servo Motor MG90S
- Relay Module
- Mini Water Pump
